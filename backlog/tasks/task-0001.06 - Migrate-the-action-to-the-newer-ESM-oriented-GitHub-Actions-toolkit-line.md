## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Track the future migration from the conservative CommonJS-compatible GitHub Actions toolkit baseline to the newer ESM-oriented toolkit line once the current dependency modernization initiative is complete. Upstream toolkit packages have moved to newer major versions with ESM packaging, newer @actions/http-client behavior, and tighter assumptions about Node 20+ tooling. This task should be treated as a follow-up initiative after the conservative dependency cleanup is landed and stable.
