## Implementation Notes

<!-- SECTION:NOTES:BEGIN -->
2026-04-08: Task started on branch `chore/task-0001-06-esm-toolkit`.

Published package metadata check before implementation:
- `@actions/core@3.0.0`, `@actions/exec@3.0.0`, `@actions/io@3.0.2`, and `@actions/http-client@4.0.0` are ESM packages (`type: module` / import-only exports).
- `@actions/tool-cache@4.0.0` depends on that full newer toolkit line.

Planned migration approach:
1. Upgrade the `@actions/*` runtime dependencies together.
2. Move the TypeScript and Jest compilation path to an ESM-compatible configuration without changing unrelated tooling in this task unless required for the toolkit migration.
3. Rebuild the bundle and validate local tests/CI commands.

If the ESM toolkit migration exposes additional unrelated modernization work that should not be bundled into this task, that work should be split into a follow-up task instead of absorbed here.

Completed on April 8, 2026.

Implemented upgrade path:
- Upgraded the runtime toolkit packages to `@actions/core@^3.0.0`, `@actions/exec@^3.0.0`, `@actions/http-client@^4.0.0`, `@actions/io@^3.0.2`, and `@actions/tool-cache@^4.0.0`.
- Moved the repository onto an ESM package boundary with `"type": "module"` in `package.json`.
- Updated TypeScript source and test imports to explicit `.js` relative specifiers so the source compiles and runs cleanly in ESM mode.
- Switched Jest to an ESM-compatible setup: `jest.config.cjs`, ESM-aware `ts-jest`, explicit `@actions/*` package entrypoint mapping for Jest resolution, `globals.cjs` setup, and `node --experimental-vm-modules` in the test script.
- Rebuilt the action bundle with `ncc`, which now emits `lib/index.js` together with `lib/package.json` carrying `"type": "module"` for the bundled output.

Validation:
- `npm install`
- `npm run tsc`
- `npm run build`
- `npm run lint`
- `env RUNNER_TEMP=/tmp npm test -- --runInBand`

No additional follow-up task was needed from this migration pass. The remaining open dependency-modernization follow-up under TASK-0001 is still TASK-0001.07 for the deprecated ESLint 7 line.
