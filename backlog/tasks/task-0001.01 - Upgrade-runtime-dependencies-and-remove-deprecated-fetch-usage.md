## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Bring the action's runtime dependency set up to date with the current Node 20.10+ runtime and eliminate runtime code that is only present to support old libraries. The main runtime dependency risk is src/get-latest-version.ts and its tests, which currently rely on node-fetch v2-specific imports and FetchError types. The task should update runtime packages in package.json/package-lock.json and clean up the fetch path so the action continues to resolve Hugo versions and install binaries correctly.
